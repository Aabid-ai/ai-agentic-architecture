# What is an AI Agent?

An AI agent is a system that can:

- Perceive information (input)
- Process or think (using logic or AI models)
- Take actions (output)

## 🔹 Simple Definition
An AI agent = Input → Thinking → Action

## 🔹 Example
- ChatGPT → answers questions  
- Google Maps → suggests routes  
- Self-driving car → drives automatically  

## 🔹 Components of an AI Agent

1. **Input**
   - User query / data

2. **Processing**
   - LLM (Large Language Model)
   - Rules / logic

3. **Output**
   - Response / action

## 🔹 Why Important?

AI agents help automate tasks like:
- Chatbots
- Automation tools
- Smart assistants
